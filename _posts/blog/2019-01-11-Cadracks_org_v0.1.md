What you can do in cadracks.org v0.1
------------------------------------

- create your user account
- create a repository
- clone the project
- add some files, commit theam and push to cadracks.org
- view CAD files in the 3D viewer

Limitations
-----------

- Limited set of 3D viewable files
- No private repos
