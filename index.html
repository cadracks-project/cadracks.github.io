CadRacks Pages

Welcome to Cadracks Pages, the home of the CadRacks project blog and documentation

CadRacks is an Open Source project which aim is to bring collaboration and decentralized version management to CAD users: innovators, engineers, designers ... working together from anywhere, at any time, using any CAD software to create 3D computerized plans and specification

You can think of CadRacks as Github, but specifically tailored to CAD

Try it at cadracks.org
